# Noob-CRUD-Node

## Features:
- Get all data
- Get single data by id
- Write data
- Update data by id (as example)
- Delete data by id (as example)

## How To Run:
```
$ git clone https://github.com/MehedilslamRipon/Noob-CRUD-Node.git

$ cd Noob-CRUD-Node

$ node server.js
```

Now open `POSTMAN` and play with `CRUD`!
